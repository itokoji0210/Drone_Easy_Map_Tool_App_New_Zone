# ドローン飛行範囲作成マップ

社内のドローン飛行申請で使う、飛行範囲図作成用の静的Webアプリです。
GitHub Pagesなどに `index.html`、`style.css`、`app.js` をアップロードすれば動作します。

## 今回の主な変更

- 地理院地図の `drone_rz_yz_2607` レイヤーをPNGタイルとして直接参照
- 「改正法 7/14〜」レイヤーの表示状態を画面右上に表示
- PDF/JPEG出力にも残る短い注意書きを追加
- 出典表記を、DID・空港周辺・小型無人機等飛行禁止法レイヤーを参考表示していることが分かる文言に修正

## ファイル構成

```text
index.html
style.css
app.js
README.md
```

## 使い方

1. GitHubのリポジトリを作成する
2. この4ファイルをアップロードする
3. GitHub Pagesを有効にする
4. スマホまたはPCのブラウザで開く

## 注意事項

このアプリは、飛行可否を自動判定するものではありません。

DID、空港周辺、緊急用務空域、150m以上、第三者上空、小型無人機等飛行禁止法、自治体・施設管理者の制限などは、必ず国土交通省、警察庁、国土地理院、SORA map、関係機関の公式情報で別途確認してください。

地図レイヤーは、配信元データの更新状況や通信状態により、最新の規制範囲を完全に表示できない場合があります。

## 改正法レイヤー

今回の実装では、地理院地図で使われている次のタイルをLeafletで重ねています。

```js
https://maps.gsi.go.jp/xyz/drone_rz_yz_2607/{z}/{x}/{y}.png
```

このため、黄・赤の区域は地理院地図の表示に近い形で重なります。ただし、アプリ側ではポリゴン判定や飛行可否判定は行いません。
